# Chat-Sphere
Chat Sphere 💬 | AI-Powered Chatbot
Chat Sphere is a simple yet powerful chatbot web application that provides intelligent responses, performs mathematical calculations, and supports multiple chat sessions. Built with HTML, CSS (Tailwind), and JavaScript, this project features a clean UI, chat history, and a smooth user experience.

🔹 Features
✔️ Interactive Chat System – Get AI-powered responses.
✔️ Mathematical Calculations – Perform basic arithmetic operations.
✔️ Multiple Chat Sessions – Save and switch between conversations.
✔️ Auto-scroll & Message Input – Send messages via the button or Enter key.
✔️ Dark Mode UI – A sleek interface with a dark background.
✔️ Profile Menu – Includes login and profile options.

📌 Tech Stack
Frontend: HTML, Tailwind CSS, JavaScript
Logic: Vanilla JavaScript
Design: Minimalistic and user-friendly
